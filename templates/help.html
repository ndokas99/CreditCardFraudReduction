CCFRS

Introduction

Is a transactional security system. It is implemented into credit card systems in order to reduce risk of fraud by altering account details after every transaction. It can be integrated into a Point Of Sale Terminal (POST) or into an E-commerce website when making payments.


Usage

Operations which are integrated into this system are:

  • Create an account to use for transactions
  • Perform transactions by using account details
  • Send feedback to the System administrators
  • Request blocking of your account
